SistemaFarmacia

Projeto simples em Java para gerenciamento de uma farmácia, com conexão a banco de dados MySQL.

Tecnologias
Java
 JDBC
 MySQL
 IntelliJ IDEA

Funcionalidades
Cadastro de medicamentos
 Listagem de medicamentos
 Conexão com banco de dados

Banco de Dados
O projeto utiliza MySQL rodando localmente.

Configuração básica:

URL: jdbc:mysql://localhost:3306/farmacia
Usuário: root

 A senha do banco não está incluída no projeto por segurança.

Como executar
1. Clone o repositório
2. Configure o banco de dados no seu computador
3. Ajuste usuário e senha no código (ou use variáveis de ambiente)
4. Execute a classe `Main.java`

Observações

Esse projeto foi feito para fins de estudo e prática com Java e banco de dados.
